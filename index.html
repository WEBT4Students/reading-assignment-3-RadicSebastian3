HTML for Lists

Question 1 : Start Tags for an ordered list, unordered list, description list, a list item, a description term, a description data

ordered list : <ol>
unordered list : <ul
description list : <dl
a list item : <li>
a description term: <dt>
a description data: <dd>

CSS for Lists

1. Make a bullet square

ul {list-style-type: square;}

2.Make bullet an image named "my-bullet.png"

ul{list-style: square inside url("my-bullet.png">

3.Make bullet an uppercase roman number

ol{list-style-type: upper roman;}

4.Set bullet position to be inside the list item's text flow

list-style-position: inside; means that the bullet points will be inside the list item.

Set bullet position to be outside the list item's text flow

list-style-position: outside;

CSS for tables

1. Black dotted border of 1 px width.

Solid border of 1 px width having a red color and rounded corners

Merge the borders of table cells so that they appear as one line
